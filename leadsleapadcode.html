Many of the tools and splash pages I have created using AI include LeadsLeap PPC ads in the code. You get paid on these ads much like Google AdSense, but with some key differences. They aren't going to pay as much per click as the mighty Google. However,  you are accepted WITHOUT having to apply and WITHOUT even having a website. You can freely put the ad code on any site that will accept the script. Github does display these ads. Here you go:




Because these ads are used so much in my products, you should have the ad code from your own LeadsLeap account ready to go. Sign up for a free account first if you don't have one yet. They also provide free list building and ad tracking. So there are plenty of good reasons to have an account even if it is a free one.

After you get an account, go to "Get PPC Widget Code" in the Earn menu. For a page like this, I choose 4 ads and "Responsive." You then just copy and paste the code where you need it. For example, I have splash page and PPC ad wizards linked to from the home page that will ask you to put in the code. So have it ready or at least know where to find it.
